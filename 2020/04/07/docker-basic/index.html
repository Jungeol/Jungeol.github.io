옵션 설명
-d 백그라운드 모드
-p 호스트와 컨테이너의 포트 포워딩 [h포트:c포트]
-v 호스트와 컨테이너의 디렉토리를 마운트
-e 컨테이너 내에서 사용할 환경변수 설정
-name 컨테이너 이름 설정
-rm 프로세스 종료시 컨테이너 자동 제거
-it -i와 -t를 동시에 사용한 것으로 터미널 입력을 위한 옵션
–link 컨테이너 연결 [컨테이너명:별칭]
